Quick Troubleshooting Guides

A list of quick troubleshooting steps.

Adding a Device to the Campus Domain

To add a device to the campus's domain, complete the following steps:

  1. Press Win+R on the device's keyboard and type sysdm.cpl. This will open the System Properties window.
  2. Click Change... to open the Computer Name/Domain Changes. window.
  3. In the Member of section of this window, make sure that the Domain is set to ust.stthom.edu. If it is not, set it to ust.stthom.edu, and be sure that the Computer name states either USTLAT01XXXX or USTOPT01XXXX (replace the XXXX with the number that is has already been assigned if necessary).
    1. Use USTLAT if the device is a laptop.
    2. Use USTOPT if the device is a desktop.
  4. Click OK. You will be prompted to enter credentials to join the computer to the school's domain. Enter your UST credentials (username and password).
    • NOTE: The .\administrator and DT password will not work when joining the domain.
  5. If your credentials have been accepted, you should receive a prompt that states, "Welcome to the ust.stthom.edu domain." If you have received an Access denied eror, temporarily use the ustsnowstudents username and password; talk to Justin Puder when you have a chance so that you have permission to add computers to the school's domain. Click OK.
  6. Click OK when prompted to restart the computer. Click Close in the System Properties window. Click Restart Now when prompted.

Using Control Panel instead of Windows Settings

If you are working in IT, you should be using Control Panel, Task Manager, and Services for tasks such as disallowing the computer to sleep, disabling/enabling startup apps, and disabling services. Windows Settings does not change everything and sometimes doesn't apply changes at all.

Disallowing Sleep in Control Panel

There is a more formal way of disallowing a computer to sleep, but this setting can be found in the BIOS of most computers on campus. However, Windows has a workaround to disallow sleep so you do not have to restart the computer to enter the BIOS to disallow sleep. To do this, complete the following steps:

  1. Click on the Windows Start Menu, type Control Panel.
  2. In the top-right search bar, type power and click Power Options.
  3. Under Selected plan, click Change plan settings for your selected power plan.
  4. The device you are working on may or may not look different compared to the image below. However, change the Turn off the display and Put the computer to sleep settings to Never.
  5. Click Save changes.

Changing Startup Applications in Task Manager

Again, while Windows Settings contains a section for changing the Startup applications, the changes you make in Windows Settings do not apply, and you will have those applications launch at login after rebooting. Complete the following steps:

  1. Click on the Windows Start Menu, type Task Manager.
  2. Click on the Startup tab within Task Manager. Here is where you will enable/disable startup applications. NOT Windows Settings.

If you notice that Adobe, Edge, etc. are launching at startup, Task Manager is where you would resolve this issue.
